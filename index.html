Performance Contracting & Principles

Performance Contracting

Performance Contract is a tool aimed at improving efficiency and effectiveness in the management of public service.

It involves four main stages;

1. Negotiation,

2. Vetting,

3. Implementation, and

4. Evaluation.

Performance Contracting Principles

The following are the Performance Contracting Principles;

1. What gets measured gets done,

2. If you don’t measure results, you cant tell success from failure,

3. If you can’t see success, you cant reward it

4. If you cant see success, you cant learn from it,

5. If you cant recognize failure, you cant correct it,

6. If you cant demonstrate results, you cant win public support
